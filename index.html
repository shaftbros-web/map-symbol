使い方・SVGの置き方
  • この版は symbols/ フォルダの SVG を読み込みます。ファイル名は data に指定した相対パスです。
  • SVGが見つからない／読み込み失敗時は fallbackChar（例：〒, 卍 など）で表示します。
  • GitHub Pages で公開する場合、この index.html と symbols/ を同じ階層に配置してください。

必要ファイル名（例） symbols/post.svg, temple.svg, shrine.svg, school.svg, hospital.svg, parking.svg, factory.svg, cityhall.svg, police.svg, station.svg, mountain.svg, orchard.svg, tea.svg, paddy.svg, field.svg, lighthouse.svg
